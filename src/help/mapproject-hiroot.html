hiroot

HiRISE PDS Root Filename (HiRISE only)

Enter the root filename for the HiRISE imagery. This name must be generic to all the input HiRISE files and should end in "RED". Example:

  • PSP_002387_1985_RED

POMM will use this filename root to find (wildcard) all the focal plane files. Only the Red band is currently supported (BG and IR are future upgrades). Leave this field blank if CTX, HRSC, or other sensor data are being processed.
