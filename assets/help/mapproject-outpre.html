PREFIX Filename for Output Products

Output filenames will start with this supplied description/prefix.

  • Example: ctx_B04_011266

The prefix should not exceed 30 characters and must follow standard Linux filename conventions (e.g., no blanks or special characters). The output map product will have "_map.tif" appended as part of the filename. For example: ctx_B04_011266_map.tif
