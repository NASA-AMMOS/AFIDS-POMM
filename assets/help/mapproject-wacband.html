wacband

LRO WAC "CC" PDS Band Number Code.

Enter the integer Band Number "code" of the input LRO WAC (Wide Angle Camera "CC" format) PDS image to-be-processed. The WAC "CC" PDS file contains "COLOR" mode files from the UV and VIS sensors. Currently only the VIS sensors are supported via wacband codes 3-7:

  • Codes:
    • 1 - UV1 (321nm) (not supported)
    • 2 - UV2 (360nm) (not supported)
    • 3 - VIS 1 (415nm)
    • 4 - VIS 2 (566nm)
    • 5 - VIS 3 (604nm)
    • 6 - VIS 4 (643nm)
    • 7 - VIS 5 (689nm)
