Map-Projecting - Finding Planetary Images for Projection

The following websites can be used to find and download raw PDS imagery. Note that the imagery needs to be copied to the Linux cpu hosting POMM. If your browser downloads to a Windows or Mac computer, the files will need to be uploaded to Linux. The general procedure is to navigate to your "downloads" directory and open a terminal (command prompt) window. Then use "cp" (copy command; or "scp" secure copy command) to upload to file to the Linux box, for example:

  • cp *.IMG "username@linux.cpu:/data/mywork/raw_image/."
  • Find and Download Mars or Luna/Moon Orbital Images:
    • USGS Pilot: https://pilot.wr.usgs.gov/
      • Interactive Mars and Earth/Moon image locator. Select raw images, view footprints, and download. Click the bottom left "page" icon for image name.
    • Orbital Data Explorer (PDS Geosciences Node): https://ode.rsl.wustl.edu/
      • Select "Mars Orbital Explorer" to find Mars CTX, HiRISE, and HRSC raw (and HRSC mapped) images, and download.
      • Select "Lunar Orbital Data Explorer" to find Moon NAC and WAC raw and mapped images, and download.
  • Other Online Image Sources (you will need to know the name of the desired file):
    • HRSC Raw (nd2) Sources: [Can replace "https:" with "ftp:""]
      • https://pds-geosciences.wustl.edu/mex/mex-m-hrsc-3-rdr-v3/mexhrs_1001/data
    • CTX/HiRISE raw PDS (Level 0) Images:
      • https://pds-imaging.jpl.nasa.gov/volumes/mro.html. (CTX and HiRISE)
      • https://pds-imaging.jpl.nasa.gov/data/mro/mars_reconnaissance_orbiter/ctx/. (CTX)
